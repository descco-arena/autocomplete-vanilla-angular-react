## Install

- [Node.js](https://nodejs.org/)
- Grunt 

```bash
npm install -g grunt-cli
npm install
```

## Usage

(automatically opens the address `http://localhost:8000`):

```bash
npm start
```

## Documentation

`http://localhost:8000/doc/`
